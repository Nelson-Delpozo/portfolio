Projects Just a few highlights of my recent work

Keg-Lister

This is an ad-lister application similar to Craigslist, but the ads are listings of various independent craft breweries in Texas. Visitors can view all the listings, create an account, login and logout, and add new listings. Logged-in users can also edit or delete their own posted ads. It’s built using Java, MySQL, JSP and JSTL, and also leverages CSS and Bootstrap for the front-end.

Movie Database

This is a Netflix-style application that allows users to view a listing of movies with poster images, cast, release date, synopsis, and reviews pulled from The Movie Database using their API. It was built using Javascript, HTML, CSS, and jQuery as well as the Bootstrap framework.

Weather and Traffic Map API Project

This application will fetch a real-time 8-day weather forecast using AJAX to receive JSON data from the OpenWeatherMap API. Using Javascript events, you can grab the latitude and longitude from the Mapbox API to fetch a location’s weather data. Or, you can use the auto-complete search bar to get a requested location. This project utilizes HTML, Javascript, Jquery, with the Bootstrap framework.
